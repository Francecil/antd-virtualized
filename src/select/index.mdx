# Select 选择器

下拉选择器。

## 引入

```
import { Select } from 'antd-virtualized';
```
## 代码演示

基本使用
<Playground>
<Select
  placeholder="支持大数据的Select"
  labelKey='name'
  options={Array.from({ length: 2000 }).map((v, i) => ({ id: i, name: 'test' + i }))}
  valueKey='id'
  style={{ width: 120 }}
/>
</Playground>

带搜索框
<Playground>
<Select
  allowClear
  showSearch
  placeholder="支持大数据的Select"
  labelKey='name'
  options={Array.from({ length: 2000 }).map((v, i) => ({ id: i, name: 'test' + i }))}
  valueKey='id'
  style={{ width: 120 }}
  filterOption={(input, option) => option.name.indexOf(input) >= 0}
/>
</Playground>

定义下拉列表为空时显示的内容
<Playground>
原始
<Select style={{ width: 120 }}/>
<br/><br/>
自定义 
<Select
  style={{ width: 120 }}
  notFoundContent="数据为空"
/>
</Playground>
